# Quiz Dev JS

Quiz interativo feito com HTML, CSS e JavaScript puro.

## Funcionalidades

- Perguntas por categoria: HTML, CSS, JavaScript e Git
- Cronômetro por pergunta
- Pontuação com bônus por velocidade
- Ranking salvo no navegador com `localStorage`
- Layout responsivo com visual moderno

## Como rodar

Abra o arquivo `index.html` no navegador.

Se quiser servir localmente:

```bash
npx serve .
```

## Tecnologias

- HTML5
- CSS3
- JavaScript
- localStorage
